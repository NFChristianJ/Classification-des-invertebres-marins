Conditions d'Utilisation

Règles d'usage de la plateforme InvertID

En utilisant InvertID, vous acceptez pleinement ces conditions.

Objet

Les présentes Conditions Générales d'Utilisation (CGU) régissent l'utilisation du service InvertID, plateforme de classification des invertébrés marins par intelligence artificielle (Reseaux de neurones).

Accès au service

2.1. L'utilisation nécessite :
- Un compte utilisateur valide (pour certaines fonctionnalités)
- Une connexion internet stable
- Un navigateur à jour (Chrome, Firefox, Edge)

2.2. Les mineurs doivent obtenir l'accord de leur tuteur légal.

Restrictions d'usage

Il est strictement interdit de :
- Utiliser le service à des fins commerciales sans autorisation
- Uploader des images contenant des données personnelles ou sensibles
- Tenter de reverse-engineer le modèle d'IA
- Surcharger intentionnellement les serveurs (DoS)
- Utiliser des bots pour automatiser les requêtes
- Publier des résultats d'identification sans mentionner le caractère probabiliste des suggestions
- Utiliser le service pour valider des diagnostics vétérinaires ou médicaux
- Réutiliser les images traitées pour entraîner d'autres modèles d'IA sans autorisation écrite

Données et confidentialité

4.1. Les images uploadées :
- Sont analysées en temps réel
- Sont stockées pour amélioration du modèle
- Ne sont pas partagées avec des tiers sans consentement

4.2. Données collectées :
- Métadonnées techniques (navigateur, résolution)
- Résultats de classification (anonymisés)
- Interactions avec l'interface (clics, temps de session)

4.3. Propriété intellectuelle des contributions :
- Les images fournies par les utilisateurs scientifiques partenaires restent leur propriété
- Les annotations expertes contribuées peuvent être utilisées pour améliorer le modèle

4.4. Cookies :
- Cookies techniques indispensables au fonctionnement (session, sécurité)
- Cookies analytiques optionnels (mesure d'audience)

Limitations techniques

5.1. Précision du modèle :
Le taux de reconnaissance moyen est de 89% mais peut varier selon :
- La qualité de l'image
- La rareté de l'espèce
- Les conditions de prise de vue (lumière, turbidité)

5.2. Disponibilité :
Service opérationnel 24h/24 sauf :
- Maintenance programmée (préavis de 48h)
- Cas de force majeure (cyberattaque, catastrophe naturelle)

5.3. Quotas d'utilisation :
- Limite de 10 requêtes/heure pour les comptes gratuits
- Priorité donnée aux partenaires scientifiques lors des pics de charge

5.4. Compatibilité :
- Non compatible avec les formats RAW professionnels
- Taille maximale des images : 10MP

Responsabilités

6.1. InvertID :
- Fournit des suggestions algorithmiques non contractuelles
- Ne garantit pas l'exhaustivité des espèces référencées

6.2. L'utilisateur :
- Doit vérifier les identifications pour les usages scientifiques
- Est responsable du contenu qu'il upload
- Doit citer InvertID dans les publications utilisant nos données

Propriété intellectuelle

7.1. Le modèle de classification :
- Architecture neuronale protégée par secret commercial
- Poids du modèle sous licence propriétaire

7.2. Base de connaissances :
- Référentiel taxonomique intégrant WoRMS (World Register of Marine Species)
- Corpus d'images sous licence CC-BY-NC pour la recherche

Usage scientifique

8.1. Publications :
- Citation obligatoire : "Classification réalisée avec InvertID (https://invertid.org)"
- Notification requise pour les usages dans des publications à comité de lecture

8.2. Collaboration :
- Programme ouvert aux laboratoires pour contribution d'images expertisées
- Accès API possible sur demande pour les institutions partenaires

Résiliation

9.1. Suspension possible :
- En cas de violation répétée des CGU
- Pour usage anormalement intensif

9.2. Conservation des données :
- 30 jours après suppression du compte pour les données personnelles
- 5 ans pour les données anonymisées à des fins de recherche

Modifications

Les CGU peuvent être mises à jour. La version en vigueur est toujours accessible sur cette page. En cas de changement majeur, les utilisateurs actifs seront notifiés par email.
