1. Objet de l'application

InvertID est une plateforme utilisant l'intelligence artificielle notament les reseaux de neurones pour classifier les invertébrés marins à partir d'images. Destinée aux chercheurs et passionnés de biologie marine, elle fournit des identifications automatiques non contractuelles.

2. Propriété intellectuelle

Modèle d'IA : Le système de classification est protégé par les droits d'auteur de l'université de Yaoundé 1. L'architecture base sur les reseaux de neurones est utilisée sous licence open-source.

Base de données : Les images d'entraînement proviennent de sources : GBIF, Kaggle, etc. avec attribution requise.

Contenu : Tous les éléments graphiques et textuels sont la propriété exclusive d'InvertID sauf mention contraire.

3. Protection des données (RGPD)

Conformément au Règlement Général sur la Protection des Données (RGPD), nous collectons :

  • Données de compte (email, nom) pour la gestion utilisateur
  • Images uploadées (stockées temporairement)
  • Cookies techniques (session, préférences)

Droits des utilisateurs : Accès, rectification, suppression via privacy@invertid.com.

4. Limitations de responsabilité

- Les identifications fournies sont des suggestions algorithmiques nécessitant une validation experte pour les usages scientifiques.
- InvertID décline toute responsabilité pour les décisions prises sur la base des résultats.
- La disponibilité du service est assurée "en l'état" sans garantie de continuité.
