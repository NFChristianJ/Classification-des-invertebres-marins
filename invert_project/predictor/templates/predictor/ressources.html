API InvertID

Version 2.1 - Dernière mise à jour : 15/06/2024

Notre API REST permet d'intégrer la classification des invertébrés marins directement dans vos applications.

Fonctionnalités clés :

  • Classification en temps réel
  • Analyse par lot (batch processing)
  • Retour des 5 meilleures correspondances
  • Métadonnées taxonomiques complètes
Limite gratuite : 1,000 requêtes/mois - Clé API requise
